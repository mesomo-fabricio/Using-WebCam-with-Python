Começar instalando o BI:
Definir:
C:\CIGAM\CIGAM11\BI
Apos descompactyar, copiar os atalhos
CIGAM BI e MANUT BI
para a pasta atalhos

Abrir Atalho MANUT BI
->BOtão Instalação 
--> Em conexao, informa Oracle, ou SQSERVER
Depois clica em "String Conexao"
Informa os dados de conexao(user, pass, IP..(Nunca localhost, por causa dos acessos via rede))


Feito isso, fechar a tela de coneaxo.
Na tela atual, caso esteja usando CIGAM, marcar a opção "Utiliza CIgam" e tbm "Mesma conexao Cigam"
- Escolher local do magic.ini
- Se tiver chave do google maps, informar
- Na caixa Instalar BI Local, nao marcar nada
- Em proxy - Marcar configs do Windows
---Criar

Proxima tela criação do usuário de administração do BI
- Administrador / manager e vincula-lo ao usuario supervisor

- Licença deve ser solicitada a Provise, clicando em solicitar licenca

APP WEB
Descompactar o arquivo WEB em C:\CIGAM\CIGAM11\BI\Web\web.config

Ir ate a pasta BI, renomear o atalho Manut BI para Manut BI - Web e move-los para a pasta atalhos
OBS: O Config.xml, presente nas propriedades do atalho MANUT BI Web só sera gerado apos o click neste atalho

Para isto,
Edite o arquivo C:\CIGAM\CIGAM11\BI\WEB\CONFIG.XML
e, na entrada
<add key="arquivoConfig" value="">  deixe o da seguinte forma
<add key="arquivoConfig" value="C:\CIGAM\CIGAM11\BI\config.xml">

Abrir novamente o Atalho e Clicar em "Atualizar"
